# Shopify Featured Products Section

Цей репозиторій містить секцію **Featured Products** для Shopify.

## Опис

Секція відображає рекомендовані продукти з обраної колекції.  
- Товари, які вже є в кошику, не показуються.  
- Додавання товару відбувається без перезавантаження сторінки (AJAX).  
- Секція адаптивна, mobile-first, кросбраузерна.  
- Використовується стандартна тема **Dawn**, без jQuery та CSS фреймворків.  
- Використані SCSS змінні для стилів та BEM неймінг.
